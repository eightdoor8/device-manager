# WEB管理画面 - PC向けレイアウト・ナビゲーション設計

## 概要
Device Manager WEB管理画面は、QA管理者向けの管理ツールです。PC操作を前提とした設計で、効率的なデータ管理と操作性を実現します。

## 画面構成

### 1. 全体レイアウト
```
┌─────────────────────────────────────────────┐
│  ヘッダー（ロゴ、ユーザー情報、ログアウト）    │
├──────────┬──────────────────────────────────┤
│          │                                  │
│ サイド   │        メインコンテンツエリア      │
│ バー    │                                  │
│          │                                  │
│ (固定)   │                                  │
│          │                                  │
└──────────┴──────────────────────────────────┘
```

### 2. ヘッダー
- **左側**：Device Manager ロゴ
- **右側**：
  - ユーザー名表示
  - ログアウトボタン

### 3. サイドバー（左側固定）
- **幅**：250px（固定）
- **背景**：ダークグレー（#2c3e50）
- **テキスト**：ホワイト

#### ナビゲーション項目
1. **ダッシュボード** (/dashboard)
   - アイコン：📊
   - 説明：統計情報と概要表示

2. **端末管理** (/devices)
   - アイコン：📱
   - 説明：端末情報の表示・編集・削除

3. **ユーザー管理** (/users)
   - アイコン：👥
   - 説明：ユーザー情報の表示・権限管理

4. **設定** (/settings)
   - アイコン：⚙️
   - 説明：システム設定

### 4. メインコンテンツエリア
- **背景**：ライトグレー（#f5f5f5）
- **パディング**：20px
- **最小幅**：calc(100% - 250px)

## 各画面の設計

### ダッシュボード
- **統計カード**：4列グリッド
  - 総端末数
  - 利用可能な端末数
  - 貸出中の端末数
  - ユーザー数

- **クイックアクション**：
  - 端末管理へのリンク
  - ユーザー管理へのリンク

### 端末管理
- **テーブルビュー**：
  - 列：端末ID、機種名、OS、ステータス、貸出者、操作
  - ソート機能：各列でソート可能
  - フィルタ機能：ステータス、OS、メーカーでフィルタ可能
  - ページネーション：20件/ページ

- **操作**：
  - 詳細表示（モーダル）
  - 編集（インラインまたはモーダル）
  - 削除（確認ダイアログ）
  - CSV出力

### ユーザー管理
- **テーブルビュー**：
  - 列：ユーザーID、メール、名前、権限、最終ログイン日時、操作
  - ソート機能：各列でソート可能
  - フィルタ機能：権限でフィルタ可能
  - ページネーション：20件/ページ

- **操作**：
  - 詳細表示（モーダル）
  - 権限変更（インラインまたはモーダル）
  - 削除（確認ダイアログ）

## レスポンシブ対応

### PC（1200px以上）
- サイドバー表示：常時表示
- テーブル表示：フル表示
- グリッド：4列

### タブレット（768px～1200px）
- サイドバー表示：ハンバーガーメニュー
- テーブル表示：スクロール可能
- グリッド：2列

### モバイル（768px未満）
- サイドバー表示：ハンバーガーメニュー
- テーブル表示：カード表示に変更
- グリッド：1列

## インタラクション

### ナビゲーション
- 現在のページはサイドバーでハイライト表示
- ホバー時に背景色が変更
- クリック時にページ遷移

### テーブル操作
- ヘッダークリック：ソート
- 行クリック：詳細表示
- アクションボタン：編集・削除・詳細表示

### モーダル
- 詳細表示、編集、削除確認をモーダルで表示
- 背景クリック：モーダルを閉じる
- キャンセルボタン：モーダルを閉じる
- 確認ボタン：アクション実行

## アクセシビリティ
- キーボードナビゲーション対応
- スクリーンリーダー対応
- コントラスト比：WCAG AA準拠
- フォーカスインジケータ：明確に表示

## パフォーマンス
- 遅延ロード：ページネーション使用
- キャッシング：tRPCのキャッシング機能を使用
- 最適化：不要な再レンダリングを避ける
