# Device Manager リリースノート v1.0.4 → v1.0.9

## 修正内容

### バグ修正

- **重複登録防止機能の実装**: 同じUUIDを持つ端末の重複登録を防止し、重複検出時にアラートを表示

- **マイデバイス表示の修正**: 借りた端末がマイデバイス一覧に正常に表示されるよう修正

- **ボタンレイアウトの改善**: 返すボタンと削除ボタンのギャップを追加し、ボタンサイズを最適化

- **ログイン時のエラーメッセージ改善**: メール形式チェックとパスワード長チェックを追加し、Firebaseのエラーメッセージが直接表示されないようにしました

- **クラッシュ問題の解消**: アプリ起動時のクラッシュ問題を修正

### 機能追加

- **端末削除機能**: 端末詳細画面に削除ボタンを追加し、ユーザーが不要な端末レコードを削除可能に

- **実務的なフィルタ機能**: OSとメーカーでフィルターがかけられるようにしました

- **検索UXの改善**: 検索バーに「×」クリアボタンを追加し、検索実行を明確化

### UIの改善

- **検索バーの改善**: クリアボタンを追加して検索機能の使いやすさを向上

- **フィルタボタンの追加**: ステータス、OS、メーカーでの多次元フィルタリングが可能に

## 既知の問題

- **フィルタ機能の一部不具合**: 「利用可」と「貸出中」のステータスフィルタが完全に機能していない場合があります。別途修正が予定されています。
